commands

mkdir: Create new folder

cd: Change Directory

LS: List information about file(s)

LS-I: List permission, owner and group information. Additional size and the date of the file

Ls-Ia: List all files, includin the secret files

pwd: Print Working Directory


pipes and redirectiom : Pipe is used to pass output to another program or utility.

Redirect is used to pass output to either a file or stream.


grep: earch file(s) for lines that match a given pattern

cat: Concatenate and print (display) the content of files

less: Display output one screen at a time
